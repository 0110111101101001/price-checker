PASSO A PASSO PARA SUBIR NO RAILWAY:

1. Vá em https://github.com e crie um repositório (pode chamar de price-checker).
2. Envie os arquivos deste projeto pra lá (pelo botão "upload files" no GitHub).
3. Vá em https://railway.app, faça login com GitHub e clique em "New Project" > "Deploy from GitHub repo".
4. Escolha o repositório.
5. Railway detecta o 'start' automaticamente e sobe o servidor.
6. Depois de pronto, acesse: https://SEU-LINK-DO-RAILWAY/verificar-precos

IMPORTANTE:
- Substitua o arquivo "serviceAccountKey.json" pelo que você baixou do Firebase.
- Este arquivo é sensível, não compartilhe com ninguém!
